## What This Lab Demonstrates

This lab demonstrates the design and implementation of a multi-router network using Variable Length Subnet Masking (VLSM). The network supports multiple LAN segments with different host requirements and enables communication between them.

## Key Skills

- VLSM subnetting and IP address planning
- Multi-router network configuration
- Interface configuration on routers and switches
- Network segmentation
- Connectivity testing and verification

## Network Overview

- Network: 10.0.0.0/8
- LAN A: 50 hosts
- LAN B: 25 hosts
- LAN C: 10 hosts
- Routers connected in triangular topology

## Key Takeaways

This lab highlights the importance of efficient IP address allocation and demonstrates how segmentation improves both network performance and security.
